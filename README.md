## 🎯 How to Use decision-os

decision-os lets you compare options based on criteria you define. Here’s a simple way to get started:

1. **Create a new decision**  
   Click “New Decision” or similar button in the app.

2. **Add options**  
   List the choices you want to compare, such as “Buy a new laptop,” “Upgrade current laptop,” or “Keep using the old one.”

3. **Set criteria**  
   Decide which factors matter most, like cost, speed, battery life, or warranty. Add each as a separate criterion.

4. **Assign weights**  
   Tell the app how important each criterion is. For example, you might say cost is twice as important as warranty.

5. **Score each option**  
   For every option, give a score for each criterion. Score from low to high based on how well the option meets that factor.

6. **Review results**  
   decision-os calculates scores for all options using the weights and scores you entered. It shows which choice ranks highest.

7. **Adjust and explore**  
   You can change weights or scores to see how results shift. This helps you understand trade-offs and pick the best decision.

## 💡 Tips for Better Decisions

- Be honest with your scores. Don’t guess numbers; think about each option carefully.
- Use meaningful weights. Higher weights mean that factor truly matters more.
- Try your scores multiple times to see if your results stay consistent.
- Save your work regularly to avoid losing data.
- Explore different decisions by duplicating existing ones and changing data.
